## コマンド

Djangoのプロジェクト作成

```
django-admin startproject [プロジェクト名] [作成ディレクトリパス]
```

Djangoのアプリケーション作成

```
cd [Djangoのプロジェクトルートディレクトリ]
python manage.py startapp [アプリケーション名]
```
